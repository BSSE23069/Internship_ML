## 📊 Dataset Used
**EmpatheticDialogues** (Facebook AI Research)
- ~25,000 conversations grounded in emotional situations
- 32 emotion categories (sad, anxious, angry, lonely, grateful, hopeful, etc.)
- Available on Hugging Face: `datasets.load_dataset("empatheticdialogues")`

---

## 🧠 Model Applied
| Component | Details |
|-----------|---------|
| Base Model | `distilgpt2` (82M parameters) |
| Fine-Tuning Framework | Hugging Face `Trainer` API |
| Tokenizer | DistilGPT2 + custom special tokens |
| Training Epochs | 3 |
| Learning Rate | 5e-5 |
| Batch Size | 8 (grad accumulation = 2) |
| Max Sequence Length | 128 tokens |

## 📈 Key Results & Findings

- **Emotion Distribution:** The dataset contains 32 emotion classes; `sentimental`, `afraid`, and `proud` are among the most frequent.
- **Utterance Length:** Most responses are 10–30 words, making DistilGPT2's 128-token window effective.
- **Training:** The model shows consistent loss decrease over 3 epochs on 2,000 training samples.
- **Perplexity:** Computed on the held-out test set to measure model quality.
- **Tone:** Emotion-conditioned prompts (`<|emotion|>sad<|user|>...`) steer the model toward gentle, empathetic language.

## ⚠️ Safety Note
This chatbot is designed for **emotional support and companionship only**. It does **not** provide clinical advice, diagnosis, or medical guidance. Users experiencing mental health crises should contact a licensed professional or call a crisis helpline (e.g., **1-800-273-8255**).
